# 025: ブキ/スペシャル選択を表レイアウトに変更 + 表示モード依存を解除

## 種別
UI改善

## 問題
1. 現在のブキ/スペシャル選択はそれぞれ横並び4つが2行で表示されているが、1P〜4Pの対応がわかりにくい。
2. ブキ/スペシャルのアイコン表示が「表示モード」（アイコン/テキスト/両方）に依存しているが、表示モードは今後のターゲット表示用であり、ブキ/スペシャル選択は常にアイコン+テキスト（ドロップダウン）で表示すべき。

## 修正箇所
- `src/components/Settings/MemoSection.tsx`

## 修正内容

### 表レイアウト
横軸: 1P, 2P, 3P, 4P
縦軸: ブキ, スペシャル

```
      1P          2P          3P          4P
ブキ   [icon▼...]  [icon▼...]  [icon▼...]  [icon▼...]
SP    [icon▼...]  [icon▼...]  [icon▼...]  [icon▼...]
```

各セルにはアイコン（選択済みなら）+ ドロップダウンを配置。

### 表示モード依存の解除
現在のコード:
```tsx
{displayMode !== 'text' && selectedId > 0 && (
  <img ... />
)}
```

これを `displayMode` に依存させず常にアイコン表示にする:
```tsx
{selectedRowId && weapon && (
  <img src={getWeaponIconPath(weapon.id)} ... />
)}
```

## 完了条件
- ブキ/スペシャルが 1P〜4P の列 × ブキ/SP の行の表レイアウトで表示されること
- アイコンが表示モード設定に関係なく常に表示されること
