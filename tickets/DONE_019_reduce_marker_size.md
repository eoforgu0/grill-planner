# 019: マーカーサイズ縮小 + ラベル表示を右側吹き出し形式に変更 + レーン幅調整

## 種別
UI改善

## 統合
本チケットは旧013（マーカーラベルを右側吹き出しに変更）を統合している。マーカーのサイズとラベル配置は相互に影響するため同時に変更する。

## 問題
1. 現在の `MARKER_SIZE = 20px` が大きすぎる。マーカーが密集した際に重なりやすい。
2. 湧きマーカーの時刻ラベル（上側）と方面ラベル（下側）がマーカーの上下に配置されている。100s の自動湧きマーカーではレーン上端からラベルがはみ出て見切れる。
3. 撃破マーカーの時刻ラベルも上側に表示されており、マーカーが密集すると重なる。

## 修正箇所
- `src/components/Timeline/coordinates.ts` — `MARKER_SIZE`, `LANE_WIDTH`
- `src/components/Timeline/SpawnMarker.tsx` — ラベル配置の変更
- `src/components/Timeline/DefeatMarker.tsx` — ラベル配置の変更
- `src/components/Timeline/RespawnConnector.tsx` — スポナー決定マーク（◇）のサイズ調整

## 修正内容

### マーカーサイズ縮小
`MARKER_SIZE` を 14〜16px に縮小する（現在の 70〜80%）。

合わせて:
- 枠線: 2px のまま（マーカーが小さくなっても枠線は視認可能にする）
- リスポーン接続線のスポナー決定マーク（◇）: 8px → 6px に縮小

### ラベル配置の変更

**湧きマーカー（SpawnMarker）**:
- 時刻ラベル + 方面ラベル: マーカーの右側に小さな吹き出し形式で表示
- 吹き出しは半透明背景の小ラベル（例: `background: rgba(255,255,255,0.85)`, `padding: 1px 4px`, `border-radius: 2px`, `font-size: 9px`）
- マーカーの左端から少し離した位置に配置

**撃破マーカー（DefeatMarker）**:
- 時刻ラベル: マーカーの右側に同様の吹き出し形式

### レーン幅の調整
ラベルがマーカー右側に出るため、`LANE_WIDTH` を必要に応じて拡大する（100〜120px 程度を検討）。ラベルはレーン外にはみ出してもよい（`overflow: visible` 済み）ので、はみ出し量を含めて適切に調整すること。

## 完了条件
- マーカーが従来の 70〜80% のサイズで表示されること
- マーカーがクリック・ドラッグ操作可能な大きさを維持していること（最低 12px）
- 全マーカーのラベルがマーカーの右側に表示されること
- 100s の自動湧きマーカーのラベルが見切れないこと
- レーン内のマーカーとラベルが重ならず視認できること
