# 026: ターゲット基準の文言変更 + 表示モードとターゲット基準の配置を近接させる

## 種別
UI改善

## 問題
1. 「ターゲット基準」の選択肢が「ブキ順/プレイヤー順」だが「ブキ/プレイヤー」が正しい。
2. 「表示 アイコン/テキスト/両方」と「ターゲット基準」はどちらも今後のターゲット表示用の設定だが、現在離れた場所にある。ターゲット基準を表示モードの右側に配置すべき。

## 修正箇所
- `src/components/Settings/MemoSection.tsx` — ボタンテキスト変更、ターゲット基準セクション削除
- `src/ScenarioView.tsx` — 設定パネル内にターゲット基準を表示モードの右に配置

## 修正内容

### 文言変更
```
ブキ順 → ブキ
プレイヤー順 → プレイヤー
```

### 配置変更
設定パネルの上部（キケン度の行）にある「表示: アイコン/テキスト/両方」の右隣に「ターゲット: ブキ/プレイヤー」を配置する。MemoSection 内からは削除。

```
キケン度: [100] %  表示: [アイコン] [テキスト] [両方]  ターゲット: [ブキ] [プレイヤー]  方面: 7 | 湧き: 3 | 撃破: 3
```

## 完了条件
- ターゲット基準の選択肢が「ブキ」「プレイヤー」になっていること
- 表示モードとターゲット基準が同じ行に並んでいること
- MemoSection からターゲット基準セクションが削除されていること
