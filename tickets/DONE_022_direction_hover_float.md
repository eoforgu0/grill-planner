# 022: 方面選択をマウスホバーで展開するフロートに変更

## 種別
UI改善

## 背景
020（方面ラベル選択式UI）は対応済みだが、操作方式をクリック展開からマウスホバー展開に変更し、選択肢を固定3択にする。020の実装を上書きする形で対応する。

## 修正箇所
- `src/components/Timeline/DirectionLabels.tsx` — 編集UI全面改修

## 修正内容

### 非ホバー時（通常表示）
方面名をテキスト表示（現状通り）。

### ホバー時
方面ラベルにマウスホバーすると、ラベルの右側にフロート（ポップオーバー）を表示:

```
┌───────────────┐
│ [左] [正面] [右] │
└───────────────┘
```

- 選択肢は「左」「正面」「右」の固定3つのみ
- ボタンをクリックすると即座に確定してフロート閉じる
- マウスがラベル領域 + フロートの外に出たらフロート閉じる
- 自由テキスト入力欄は不要（削除）

## 完了条件
- 方面ラベルにマウスホバーすると「左」「正面」「右」のボタンフロートが表示されること
- ボタンをクリックするとその方面名が設定されフロートが閉じること
- マウスが離れるとフロートが閉じること
- クリックで開くテキスト入力モードは削除されていること
