# 024: ブキ/SPドロップダウンの幅を最長名に合わせて統一する

## 種別
UI改善

## 問題
現在のドロップダウン幅が `w-32`（128px）固定のため、長い名前のブキ/スペシャル（例: 「クマサン印のストリンガー」12文字、「メガホンレーザー5.1ch」12文字）が途切れる。

## 修正箇所
- `src/components/Settings/MemoSection.tsx`

## 修正内容
ブキ全てとスペシャル全ての中で最も横幅の長いものに合わせた統一幅を設定する。

ブキの最長: 「クマサン印のストリンガー」「クマサン印のマニューバー」「クマサン印のチャージャー」等（12文字）
スペシャルの最長: 「メガホンレーザー5.1ch」（12文字）、「トリプルトルネード」（9文字）

12文字の日本語 + 英数を表示するには text-xs（12px）で約 160〜180px 必要。

### 実装方針

固定値で対応する:
```tsx
// ブキ・スペシャル共通のドロップダウン幅
const SELECT_WIDTH = 'w-44'; // 176px — 12文字の日本語ラベルに十分
```

ブキの select と スペシャルの select に同じ幅クラスを適用する:
```tsx
<select className="w-44 rounded-sm border ..." >
```

もし `w-44` で不足する場合は `w-48`（192px）にする。表形式の見た目を崩さないため、ブキとスペシャルは必ず同じ幅クラスを使うこと。

## 完了条件
- 最長のブキ名・スペシャル名が途切れずに表示されること
- ブキとスペシャルのドロップダウン幅が揃っていること
- 4枠すべてでドロップダウン幅が統一されていること
