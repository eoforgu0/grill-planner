# 027: タマヒロイメモのテキストボックス幅をシナリオコードと同等に縮小

## 種別
UI改善

## 問題
現在のタマヒロイメモは `w-full`（親幅いっぱい）だが、入力予定は「右前ヒロイ」等の10文字未満。シナリオコードと同等の幅で十分。

## 修正箇所
- `src/components/Settings/MemoSection.tsx`

## 修正内容
```tsx
// Before
className="w-full ..."

// After
className="w-48 ..."  // シナリオコードと同じ幅
```

## 完了条件
- タマヒロイメモのテキストボックスがシナリオコードと同程度の幅であること
