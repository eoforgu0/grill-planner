# 018: キケン度入力をスライダー廃止・テキスト入力のみにする

## 種別
UI改善

## 問題
キケン度の入力にスライダー（range）とテキスト入力（number）の2つがあり、さらにスライダーの右に `100%` というテキストが表示されていて冗長。テキスト入力だけで十分。

## 修正箇所
- `src/components/Settings/HazardLevelInput.tsx`

## 修正内容
1. スライダー（`<input type="range">`）を削除
2. 数値入力の右に `%` を表示（`100%` ではなく入力フィールドの隣に `%` 単位表示）
3. 数値入力は従来通り min=20, max=333 をバリデーション

修正後のイメージ:
```
キケン度: [100] %
```

## 完了条件
- スライダーが表示されないこと
- テキスト入力と `%` 表示だけのシンプルなUIになっていること
- 20〜333 の範囲バリデーションが機能すること
