# 010: 右ペインをタイムラインと独立した固定サイドバーにする

## 種別
UI改善（レイアウト変更）

## 問題
現在は `[header] [settings] [timeline + stats]` という縦積み構造で、統計パネルがタイムラインと同じ flex コンテナ内にある。タイムラインのスクロールに統計パネルが影響を受けたり、タイムラインの縦スペースを十分に使えない。

## 修正内容
全体レイアウトを以下の構造に変更する:

```
[header                              ]
[                        | right pane]
[  timeline area         | (stats等)  ]
[  (設定パネルも含む)      |           ]
[                        |           ]
```

具体的には:
- 右ペイン（統計パネル）はヘッダー直下から画面下端まで固定幅で常に表示
- 左側にタイムライン + 設定パネル
- 右ペインはタイムラインとは独立してスクロール可能（overflow-y: auto）

## 修正箇所
- `src/ScenarioView.tsx` — レイアウト構造の変更

## 完了条件
- 右ペインがヘッダー下〜画面下端まで固定幅で常時表示されること
- タイムラインのスクロールが右ペインに影響しないこと
- 右ペイン内は独自にスクロール可能なこと
