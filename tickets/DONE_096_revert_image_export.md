# 096: 画像出力機能のリバート（094・095の取り消し）

## 種別
リバート

## 概要

チケット094（画像出力機能）および095（画像出力の改善）で追加した全変更をリバートし、チケット093完了直後の状態に戻す。

画像出力はWebコンポーネント転用方式で再設計・再実装する予定のため、現在の独自描画方式を一旦撤去する。

## 手順

### git 操作によるリバート

094・095のコミットを特定し、新しい方から順に `git revert` する:

```bash
# 1. 094・095に該当するコミットハッシュを特定
git log --oneline

# 2. 新しい方（095）から順にリバート
git revert <095のコミットハッシュ> --no-edit
git revert <094のコミットハッシュ> --no-edit

# 3. html2canvas が package.json から除去されていることを確認し、node_modules を更新
npm install

# 4. ビルド・lint 確認
npm run lint
npm run build
```

094・095が各1コミットでない場合（複数コミットに分かれている場合）は、
093のコミット以降の094・095関連コミットを全て特定し、新しい順に revert する。

### 注意
- 094・095以外のコミット（086〜093等）には触れないこと
- revert コミットのメッセージは `Revert "094: ..."` 等、git revert のデフォルトで良い

## 完了条件
- 「画像出力」ボタンがヘッダーに存在しないこと
- `src/components/ImageExport/` ディレクトリが存在しないこと
- `src/utils/imageExport.ts` が存在しないこと
- `html2canvas` が package.json に含まれないこと
- `npm run lint` がエラーなく通ること
- `npm run build` がエラーなく通ること
- 既存機能が全て正常に動作すること
